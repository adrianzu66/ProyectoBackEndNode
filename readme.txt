- abrir la consola de node js
- ubicarse en la carpeta del proyecto ejm (cd ubicacionActual/ProyectoBackEndNode/)
- Ejecutar el servidor de la siguiente manera:
	node Server/

- Si todo se ejecuto correctamente se debe visualizar en la consola de node el siguiente mensaje:
	"Buscador listo en puerto: 8082"

-Acceder en el navegador a la siguiente direccion:
	http://localhost:8082/
